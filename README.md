# Projeto Verificador de Senha 🔐 
Simulação de autenticação por senha desenvolvida em HTML, CSS e JavaScript para utilização em um Escape Room.

## Objetivo

Simular o acesso a um sistema por meio da validação de uma senha.

Este projeto foi desenvolvido para fins acadêmicos, com foco na prática de manipulação de strings utilizando JavaScript e na construção de uma interface simples com HTML e CSS.

> **Observação:** o projeto não possui responsividade completa, tendo sido desenvolvido para utilização em notebooks.

---

## Contexto

O projeto surgiu da necessidade de validar a resposta de um enigma em um Escape Room.

Durante a atividade, os participantes deveriam reunir pistas espalhadas pelo ambiente para descobrir a senha correta. Após encontrá-la, bastava inseri-la no sistema para obter acesso, o QR code final foi utilizado para a avaliação da atividade.

## Funcionalidades ⚙

- Validação de senha.
- Tratamento de strings.
- Conversão automática do texto para letras maiúsculas.
- Exibição de mensagens para senha correta, incorreta ou campo vazio.
- Interface simples desenvolvida com HTML e CSS.

---

## Tecnologias utilizadas

- HTML5 — Estrutura da aplicação.
- CSS3 — Estilização da interface.
- JavaScript — Validação da senha e manipulação do DOM.
